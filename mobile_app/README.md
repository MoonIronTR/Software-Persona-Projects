# Mini Katalog Uygulaması

## Proje Adı
Mini Katalog Uygulaması (Mini Catalog App)

## Kısa Açıklama
Bu proje, 5 günlük Flutter eğitim müfredatının proje çıktısıdır. Proje, Flutter'da temel widget kullanımı, sayfa geçişleri (Navigator), GridView ile ürün listeleme ve basit bir sepet state yönetimini içerir. Herhangi bir ekstra paket kullanılmamış olup, JSON verileri lokal bir dosyadan okunarak (fromJSON) simüle edilmiştir.

## Kullanılan Flutter Sürümü
Projeyi çalıştırabilmek için en az **Flutter 3.x** ve **Dart 3.x** sürümü gerekmektedir. 
Proje herhangi bir ekstra bağımlılık (paket) içermemektedir. SDK içerisinde gelen material ve convert kütüphaneleri kullanılmıştır.

## Çalıştırma Adımları

1. Repoyu bilgisayarınıza klonlayın veya indirin:
   ```bash
   git clone <repository-url>
   ```
2. Proje dizinine gidin:
   ```bash
   cd mobile_app
   ```
3. Gerekli paketleri indirin (Gerçi ekstra bir paket yok ancak standart prosedür):
   ```bash
   flutter pub get
   ```
4. Uygulamayı bir emülatörde, bağlı fiziksel cihazda veya tarayıcıda çalıştırın:
   ```bash
   flutter run
   ```

Ayrıca **Visual Studio Code** veya **Android Studio** kullanıyorsanız, `lib/main.dart` dosyasını açıp doğrudan `Run` / `Debug` butonuna basarak projeyi çalıştırabilirsiniz.
